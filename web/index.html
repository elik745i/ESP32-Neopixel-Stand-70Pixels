Light

WLED-style Effect Preset Pick an effect, colors, and brightness.
Recent light presets are stored locally in this browser. Intensity is disabled because the current WS2812FX engine in this build does not expose a matching runtime control.

The LOLIN S3 Mini default build uses the board's built-in NeoPixel on GPIO10. If you are driving an external strip instead, select a different GPIO, wire that pin to DIN through a 330-470 ohm resistor, share ground with the strip power supply, and add a bulk capacitor across the strip power rails. The selected GPIO is saved in NVS and applied after settings save.

Device

When enabled, the notifier estimates charge from battery voltage, enters deep sleep at or below the threshold, turns the OLED off, then wakes on the configured timer to briefly reconnect and check network activity before sleeping again while battery remains low. Set wake interval to 0 to stay asleep until a manual wake or power cycle.
